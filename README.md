## Installation
Add this line to your application's Gemfile:

```ruby
gem 'spree_digital_product'
```

Install the gem using Bundler:
```ruby
bundle install
```

And then execute:
```bash
$ bundle exec rails g spree_digital:install
```

Or install it yourself as:
```bash
$ gem install spree_digital_product
```
